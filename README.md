# Пульт охраны банка
Запускает сервер с информацией о посещениях хранилища.

На сервере можно увидеть активные пропуска, историю посещения хранилища и длительность пребывания в хранилище для каждого пропуска. Так же узнать кто в данный момент находиться в хранилище и длительность пребывания.

## Как установить

Вам потребуются следующие данные: 

1. `DB_HOST` - Адрес базы данных.
2. `DB_PASSWORD` - Пароль доступа пользователя к базе данных.
3. `WEBSITE_SECRET_KEY` - Секретный ключ сайта.
4. `DB_PORT` - Номер сетевого порта.
5. `DB_NAME` - Название базы данных.
6. `DB_USER_NAME` - Имя пользователя для аутентификации.
7. `DEBUG_MODE` - Отладочный режим. Присвоить True/False для включения/выключения.

Все значения присвоить соответвующим переменным окружения в файле .env

Серевер запускается командой:
```
python manage.py runserver 0.0.0.0:8000
```

Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```
